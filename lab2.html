Introduction

In this experiment, you will be able to control and operate a stepper motor. You will also learn the basics of how a stepper motor functions and how the motor can be used for applications that require a motor to run for only a set period of time. The software you will be using is called Connected Components Workbench (CCW) and allows you to operate the motor using a Micro820 Programmable Logic Controller (PLC). Compared to the other motors featured in this training set-up this stepper motor has an integrated drive attached to the back of the motor. Therefore, there is no other hardware needed to control the motor other than the PLC. This experiment also makes use of user-defined function blocks (UDFB) which are pre-programmed blocks that will allow you to send commands to the Micro820 PLC. These commands can change certain modes of how the motor operates, like if it is spinning in the clockwise or counter-clockwise direction. All instructions needed to operate CCW are provided in this experiment.
