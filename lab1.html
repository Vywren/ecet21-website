Introduction

In this experiment, you will be able to control and operate a servo motor. You will learn how a servo motor operates and can be used for precision applications. You will also learn how a servo motor can be preprogrammed to rotate to set positions. The software you will be using is called Connected Components Workbench (CCW) and allows you to operate the motor using a Kinetix 5100 Servo Drive and a Micro820 Programmable Logic Controller (PLC). The Micro820 PLC is used to control the Kinetix drive and the drive is used to control and supply power to the motor. We can program the drive so that when it receives a digital input, it will operate the motor to move to our preprogrammed positions. All instructions needed to operate CCW are provided in this experiment.
