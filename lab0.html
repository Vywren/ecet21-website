Introduction

Connected Components Workbench is a program that allows us to control our motors and drives. It comes with a large amount of tools that will help with any motor related project. In this lab, we will go over how to install Connected Components Workbench

Procedure

  1. To start with, go to the Rockefeller automation page for Connected Components workshop. You can find it on google or click the link Here


  2. Click the dropdown and select all versions, then pick the latest version. Then check the box that says select files. If you have not signed in to a Rockefeller account, it wil then prompt you to sign in. If you do not have an account, complete the steps under register and repeat the earlier steps. You should arrive at a page that looks like this


  3. For our purposes we will only need to check the top box, "Connected Components Workbench Standard Multiple Languages Edition". However, feel free to try out any of the add-ons. When you finish checking your add-ons click the shopping cart button on the top right. On the next page click download now and agree to the terms and conditions. If your download has not started click the button that says direct download and click all of the links that appear.

  4. Now open the application file that you just downloaded and wait. If you downloaded the files directly skip this step.

  5. Open the application called "22.00.00-CCW-INT-Std-DVD.part1.exe". If you didn't change the install folder it will be in a folder labeled "RA". Opening it will extract to the "RA" folder by default


  6. Running the "22.00.00-CCW-INT-Std-DVD.part1.exe" application will create a folder called "22.00.00-CCW-INT-Std-DVD". inside of this folder you will find an application called "common install". Run this applicatation and follow the instructions. You will need administrator privileges. You may run into an issue where you do not have .net Framework 3.5 installed, which is a requisite for this program to run. You may be able to remedy this by opening "windows features" in control panel and checking .net Framework 3.5 before pressing ok. Then follow the instructions that pop up


Intended Results

Congratulations! You have succesfully installed Connected Components Workbench. Now you shoud be able to complete any of the following labs
